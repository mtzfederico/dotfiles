# dotfiles

## To install ##
```
git clone https://github.com/mtzfederico/dotfiles.git
cd dotfiles
./install.sh
```

## Functions ##
To compress a folder with tar and gzip
```
targz {folder}
```

To decompress a folder with tar and gzip
```
untargz {folder.tar.gz}
```

Alias to open the nginx logs (/var/log/nginx)
```
nginxlogs
```
